# ⌨️ SnowFoxOS — Alle Shortcuts

`Mod` = Windows/Super Taste

## Apps starten

| Shortcut | Aktion |
|---|---|
| `Mod + Return` | Terminal (Foot) |
| `Mod + B` | Firefox ESR |
| `Mod + E` | Dateimanager (Thunar) |
| `Mod + C` | VSCodium |
| `Mod + Z` | Taschenrechner |
| `Mod + R` | App-Launcher (Wofi) |
| `Mod + S` | SnowFox Store |
| `Mod + N` | WiFi Manager |
| `Mod + ,` | Power Menu |

## Fenster-Management

| Shortcut | Aktion |
|---|---|
| `Mod + Q` | Fenster schließen |
| `Mod + F` | Vollbild |
| `Mod + Space` | Floating umschalten |
| `Mod + H/J/K/L` | Fokus (links/unten/oben/rechts) |
| `Mod + Pfeiltasten` | Fokus (Pfeiltasten) |
| `Mod + Shift + H/J/K/L` | Fenster verschieben |
| `Mod + V` | Split horizontal |
| `Mod + Shift + V` | Split vertikal |
| `Mod + W` | Layout: Tabs |
| `Mod + T` | Layout: Split umschalten |
| `Mod + Shift + R` | Resize Modus |

## Workspaces

| Shortcut | Aktion |
|---|---|
| `Mod + 1–9` | Workspace wechseln |
| `Mod + Shift + 1–9` | Fenster zu Workspace |
| `Mod + Tab` | Nächster Workspace |
| `Mod + Shift + Tab` | Vorheriger Workspace |

## System

| Shortcut | Aktion |
|---|---|
| `Mod + X` | Bildschirm sperren |
| `Mod + Shift + C` | Sway Config neu laden |
| `Mod + Shift + E` | Power Menu |
| `Print` | Screenshot (ganzer Bildschirm) |
| `Mod + Print` | Screenshot (Bereich auswählen) |

## Lautstärke & Helligkeit

| Taste | Aktion |
|---|---|
| `XF86AudioRaiseVolume` | Lauter |
| `XF86AudioLowerVolume` | Leiser |
| `XF86AudioMute` | Stumm |
| `XF86MonBrightnessUp` | Heller |
| `XF86MonBrightnessDown` | Dunkler |

## Resize Modus (`Mod + Shift + R`)

| Taste | Aktion |
|---|---|
| `H / Links` | Schmaler |
| `L / Rechts` | Breiter |
| `K / Oben` | Kleiner |
| `J / Unten` | Größer |
| `Escape / Enter` | Beenden |
